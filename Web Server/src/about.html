About the Web Server Project

This project is a custom-built web server developed in C to handle HTTP requests efficiently on Windows. The server listens on port 9090, processes client requests, and serves static files such as HTML, CSS, JavaScript, images, PDFs, and audio/video files. It is designed with key features like:

  • Multi-threading for handling multiple client requests simultaneously
  • Error handling for invalid requests and missing files
  • Support for multiple file types (e.g., .html, .css, .js, .jpg, .png, .mp4, .mp3, .pdf)
  • Optimized performance through caching and efficient request handling
    • The project also involved overcoming various compilation errors, linker issues, and missing dependencies, leading to a deeper understanding of network programming and system-level debugging.

    Technologies Used

    • Programming Language: C
    • Networking: Windows Sockets (Winsock2)
    • Operating System: Windows
    • Methods: GET
    • Protocols: HTTP
    • Security: Not Implemented
    • Multithreading: _beginthread for handling multiple clients
    • File Handling: Efficient file serving for multiple MIME types
